Vanilla JavaScript

Vanilla JavaScript refers to the use of core, plain JavaScript without the addition of external libraries or frameworks like React, Angular, Vue, or jQuery. It represents the language in its pure, native form, leveraging only the functionalities built directly into JavaScript and the browser's native Web APIs.

React
